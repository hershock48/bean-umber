This directory contains all image assets for the Be A Number website.

- `founder/secondary-woman-sewing.jpg` - Secondary image mid-page: Close-up of young woman writing/measuring at sewing machine

### Homepage (`/`)

### Impact Section (Homepage)

### Governance Page (`/governance`)
- `governance/planning-notebook.jpg` - Optional header image: Planning notebook image (woman writing in notebook)

## Image Guidelines

- Favor full-width images for hero sections
- Maintain natural colors (no heavy filters)
- Crop for dignity and clarity
- No stock photography anywhere
- All images should be optimized for web (JPEG format recommended)
- Use descriptive alt text for accessibility
